# Word List Plugin — Плагин автоформата перечислений

VBA-плагин для Microsoft Word, который автоматически обнаруживает
все виды перечислений в документе.

## Состав проекта

- `Module1.bas` — DetectLists: обнаружение стандартных Word-списков
- `Module2.bas` — DetectPlainText: обнаружение плейн-текстовых перечислений
- `Module6.bas` — ListBlock: объединённый модуль, формирует структуру данных
- `UserForms/UserForm1.frm` + `UserForm1.frx` — форма отображения результатов
- `test_word_lists.docm` — тестовый документ со всеми типами списков
- `Задания/` — набор из 10 тестовых документов

## Установка

1. Открой Microsoft Word
2. Нажми **Alt + F11** — откроется редактор VBA
3. В меню выбери **File → Import File**
4. Импортируй по очереди: `Module1.bas`, `Module2.bas`, `Module6.bas`, `UserForm1.frm`
   > `.frx` импортировать отдельно не нужно — он подтягивается автоматически вместе с `.frm`
5. Закрой редактор VBA

## Использование

1. Открой любой документ Word с перечислениями
2. Запусти макрос одним из способов:
   - **Alt + F8** — откроется список макросов, выбери нужный и нажми **Выполнить**
   - **Alt + F11** — откроется редактор VBA, открой нужный модуль и нажми **F5** или **Run → Run Sub**
3. Доступные макросы:
   - `DetectLists` — поиск стандартных Word-списков
   - `DetectPlainText` — поиск плейн-текстовых перечислений
   - `BuildListBlocks` — объединённый анализ всех списков
4. Результат появится в окне формы
